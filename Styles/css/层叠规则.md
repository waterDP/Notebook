# css 权重

1. !important,加在样式属性值后，权重值为 10000
2. 内联样式，如：style=""，权重值为 1000
3. ID 选择器，如：#content，权重值为 100
4. 类，伪类和属性选择器，如：content、:hover 权重值为 10
5. 标签(元素)选择器和伪元素选择器，如：div,p,::before 权重值为 1
6. 通用选择器(\*) 、子选择器（>）、相邻选择器（+）、同胞选择器（~）、:not() 权重值为 0
