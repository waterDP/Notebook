🏗️ 15. HyDE(假设性文档嵌入)
■ 核心思想
拿到 query 后不直接检索,先让 LLM "假设" 一篇能回答这个问题的文档(假想文档),然后用这篇假想文档的 embedding 去检索真实文档。 因为假想文档的语义空间更接近真实答案,能突破 query 和 answer 之间的"语义鸿沟"。
■ 原理
代码
Query: "LoRA 和全参微调谁更好?"
↓  LLM 生成假想文档
"LoRA 是一种参数高效微调方法,它通过低秩分解..."
↓  用假想文档的 embedding 检索
[真实文档 1] [真实文档 2] [真实文档 3]  ← 更精准
