🎯 8. Rerank(重排序)
■ 核心思想
先用轻量级检索(向量、BM25)粗筛一轮,拿到候选集,再用一个专门的 Reranker 模型对候选集精细排序。 向量检索的"语义相似"≠"真正满足需要的答案",Reranker 能拉回排序的精度。
■ 原理
代码
Query
↓
┌───────────────────────┐
│   Step 1: 粗筛        │  ← 向量检索,召回 top-50
│   (Bi-encoder)         │
└────────┬──────────────┘
↓
┌───────────────────────┐
│   Step 2: 精排        │  ← Reranker,从 top-50 选出 top-3
│   (Cross-encoder)      │  ↑ 这里更慢但更准
└────────┬──────────────┘
↓
LLM 回答
